- `docs/tools-and-scripts.md`: What each tool/script does and when to use it.

## Quick Start
1. Run `python tools/setup_env.py` to generate `.env` defaults.
2. Install dependencies: `python -m pip install -r requirements.txt`.
3. Run CLI: `python Lyco.py --help` or `python -m lyco --help`.
4. Run tests: `python -m unittest discover -s tests`.

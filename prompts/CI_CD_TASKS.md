# CI/CD Tasks Checklist

Use this checklist before any release, merge to main, or CI/CD handoff.

1. Check for code comment needs: `prompts/01_check_comments.md`
2. Check for pylint problems: `prompts/02_check_pylint.md`
3. Check for code vulnerabilities: `prompts/03_check_vulnerabilities.md`
4. Check SBOM vulnerabilities: `prompts/04_check_sbom.md`
5. Check the tests: `prompts/05_check_tests.md`
6. Run the tests: `prompts/06_run_tests.md`
7. Validate the tests: `prompts/07_validate_tests.md`
8. Update project versions: `prompts/08_update_versions.md`
9. Update package information: `prompts/09_update_package_info.md`
10. Update project context/state: `prompts/10_update_project_state.md`
11. Update dev tooling (VS Code tasks/debug, GitLab CI, related files): `prompts/11_update_dev_tooling.md`
12. Check for secrets and forbidden content: `prompts/12_check_secrets.md`
13. Integration + config audit (whole repo): `prompts/13_integration_audit.md`
14. Post CI/CD remediation workflow: `prompts/POST_CI_CD_TASKS.md`
